  • Mobile friendly - Responsive
  • Secure for visitors - SSL

2.2 Domain name

  • Stick with .com
  • Make it brandable
  • Easy to pronouce
  • Use domain generators (https://www.nameboy.com/)

2.3 Selecting a designer or developer

  • Do you have examples?
  • What's included?
  • What's the process.
  • Measuring results?
  • Ongoing support?

2.4 Making your website convert

  • Clear value proposition
  • One clear call to action
  • Use short forms (eg: only ask email for signup)
  • Use landing pages
  • Run A/B tests

2.5 Creating engaging web copy

  • Speak to one person
  • Focus on benefits
  • Make it skimmable
  • Use visuals
  • Hire a professional
